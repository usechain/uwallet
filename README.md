#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# build mac app
npm run build:mac

# build windows app
npm run build:win

# build linux app
npm run build:linux

# build  app
npm run build:app

# init app (todo)
make init

# build android app (todo)
make android

# build ios app (todo)
make ios


```
